Welcome to the API Documentation

This is the documentation page for the API server. You can find information about available endpoints and how to use them below.

API Endpoints

  • GET /api/orders - Retrieve a list of all orders. Try it
  • POST /api/orders - Create a new order. Try it
  • POST /api/orders - Delete an order by ID. Try it
  • GET /api/products - Retrieve a list of all products. Try it
  • GET /api/products/order/:orderId - Retrieve products for a specific order. Try it
  • POST /api/products - Create a new product. Try it
  • DELETE /api/products/:productId - Delete a product by ID. Try it

For more details and usage examples, refer to the API documentation.
